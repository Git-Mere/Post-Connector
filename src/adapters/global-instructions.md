# 전역 지시사항 (모든 어댑터 공통)

> 이 파일은 모든 플랫폼 어댑터에서 공통으로 적용된다. 어댑터별 `instructions.md` 가 충돌 시 우선한다.

## 톤

- 과장 금지. 실제로 한 일만 적는다.
- "혁신적", "획기적" 같은 마케팅 단어 지양.
- 1인칭 작성. 팀 프로젝트면 본인 역할 명시.

## 사실 검증

- `ProjectData` 에 없는 수치/기능을 만들어내지 않는다.
- 사용자 보강 입력에 없는 임팩트 수치 추측 금지.

## 언어

- 사용자가 별도 지정 없으면 README 언어 기준으로 작성.
